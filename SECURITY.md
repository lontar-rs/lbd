# Security Policy

## Reporting a Vulnerability
- Please email: security@lontar.rs (or open a private security advisory on GitHub).
- Include: affected versions/commit, reproduction steps, impact assessment, and any logs.
- We aim to acknowledge within 5 business days.

## Scope
- LBD backend (Rust/Axum), migrations, and search integration.
- Public web UI and CI/CD configurations.

## Out of Scope
- Third-party dependencies (report upstream where appropriate).
- Content errors in dictionary entries (report as regular issues).

## Handling Process
1. Triage and confirm impact.
2. Prepare fix in a private branch if needed.
3. Release patched version and disclose responsibly.
